# VitalCheck

אפליקציית מעקב תקפים לציוד רפואי בתיק כונן.

## שימוש

1. פתח את `vitalcheck.html` בכרום או אדג'
2. לחץ "התחלה חדשה" ליצירת קובץ נתונים חדש, או "יש לי כבר קובץ"
   לפתיחת קובץ קיים
3. הוסף פריטים ותקפים דרך הממשק

## תכונות

- שמירה אוטומטית של הנתונים לקובץ מקומי
- צביעה אוטומטית לפי דחיפות התוקף (ירוק/צהוב/כתום/אדום)
- סינון וחיפוש
- ניהול קטגוריות
- הדפסה / ייצוא PDF
- עובד במחשב עם כרום או אדג' (לא נתמך בספארי)
